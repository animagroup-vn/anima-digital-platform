# ANIMA Digital Platform

Website V1.0 for ANIMA Group — Next.js / React / TypeScript.

## Deploy
1. Upload this project to GitHub repository:
   `https://github.com/animagroup-vn/anima-digital-platform`
2. Connect GitHub to Vercel.
3. Add domain:
   `animagroup.vn`
   `www.animagroup.vn`
4. In Cloudflare DNS, add Vercel records when Vercel provides them.
